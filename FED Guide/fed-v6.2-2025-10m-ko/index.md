## FED 시작하기
### FED 소개

<b>FED(Fasooo Enterprise DRM)</b>는 문서 파일을 암호화해 사용자가 허용된 범위 내에서만 문서를 사용하도록 제어하는 데이터 중심의 기업용 문서 보안 솔루션입니다. **FED**를 도입한 조직은 내부자에 의한 정보 유출뿐만 아니라 지능형 지속 공격(APT) 위협에도 대처할 수 있습니다.

**다양한 암호화 정책**
* 애플리케이션 기반의 자동 암호화 기능을 통해 문서가 생성되는 대로 문서를 즉시 암호화할 수 있습니다.
* 관리자는 사용자 PC내의 모든 문서를 일괄 암호화할 수 있습니다. 

**보안 문서의 사용 권한 제어** 
* 문서의 중요도에 기반해서 문서 등급을 정의하고, 문서 등급 및 문서 사용 환경(문서 소유 상태 또는 로그인 상태)에 따라 부서/사용자별로 문서 사용 권한을 다르게 설정할 수 있습니다.
* 사용 로그와 통계를 제공하므로 데이터에 기반해서 효율적인 문서 보안 정책 수립 및 조정 또는 개별 맞춤화가 가능합니다.

**실시간 정책 적용**
* 관리자 콘솔에서 설정한 정책은 사용자에게 실시간으로 반영됩니다.
* 사내 정보 시스템을 통해 다운로드 한 보안 문서에 대한 사용 권한이 실시간으로 반영됩니다.
* 정책 적용과 관련된 주요 알림 메시지가 사용자에게 실시간으로 전달됩니다.
 
**안전한 암호 키 관리**
* 인증서 기반의 키 관리 기능을 제공합니다. 
* 관리자의 모든 행위를 추적 및 관리할 수 있습니다.
* 국/내외 보안 규정을 충족합니다.

**조직도 기반의 빠른 현황 조회** 
* 부서/사용자 별로 다양한 유형의 통계 데이터를 조회할 수 있습니다.
* 문서 사용 및 라이선스 사용 현황 등 주요 데이터를 한 눈에 볼 수 있는 대시보드를 제공합니다. 

**편리한 보안 문서의 외부 공유**
* 보안 문서의 외부 공유 기능을 제공합니다.
* 외부에 공유된 문서별로 수신자의 문서 사용 권한을 개별 관리할 수 있습니다.

### DRM 클라이언트 설치 및 제거 
#### DRM 클라이언트 설치
관리자로부터 전달받은 설치 파일 링크 또는 기타 배포 방식(PMS 또는 GPO)을 통해 PC용 DRM 클라이언트 프로그램을 설치할 수 있습니다. 프로그램 설치를 시작하기 전, 서버와의 네트워크 연결 상태 및 Windows 권한 상태(프로그램 설치를 위해서는 관리자 권한으로 로그인해야 합니다)를 확인해 주시기 바랍니다.
#### DRM 클라이언트 제거
작업 표시 줄의 시작 버튼을 클릭해 모든 앱 목록으로 이동한 후 Fasoo를 클릭해 DRM 클라이언트를 제거합니다. 제거 완료 시 DRM 클라이언트가 앱 목록에서 제거되었는지 확인합니다. 
참고: 제품 설치 환경에 따라 표시되는 프로그램 이름 및 경로는 달라질 수 있습니다.
#### DRM 클라이언트 복구
FED 사용에 문제가 발생할 경우 응급 복구를 통해 소프트웨어를 복원할 수 있습니다. 작업 표시 줄의 시작 버튼을 클릭해 모든 앱 목록으로 이동한 후 Fasoo를 클릭해 DRM 응급 복구를 실행합니다. 
참고: 제품 설치 환경에 따라 표시되는 프로그램 이름 및 경로는 달라질 수 있습니다.
### DRM 클라이언트 로그인 및 로그아웃
#### DRM 클라이언트 로그인
1.	DRM 클라이언트 설치가 완료되면 로그인 창이 표시됩니다. 
2.	사용자 ID 및 비밀번호를 입력하고 로그인을 클릭합니다.
* 사용자 ID 저장을 선택하면 입력한 사용자 ID가 저장되어 다음 로그인 시 표시됩니다.
* 자동 로그인을 선택하면 입력한 사용자 lD와 비밀번호가 모두 저장되어 로그인이 필요한 경우 자동으로 로그인됩니다.
3.	시스템 트레이에서 DRM 클라이언트 아이콘이 ![](2025-12-12-16-03-39.png)로 변경되었는지 확인합니다. 
참고: 
1. 제품 도입 환경에 따라 로그인 창 형태는 달라질 수 있습니다. 또한, DRM 클라이언트는 SSO (Single Sign On)를 지원하므로 FED 외 파수의 다른 제품을 도입한 경우에도 통합 로그인이 가능합니다. 따라서, 도입한 여러 제품의 사용을 위해 사용자 ID와 비밀번호를 동일하게 유지하십시오.
2. 비밀번호 유효 기간이 설정된 경우, 비밀번호 유효 기간이 만료되면 임시 비밀번호가 발급되어 등록된 전자 메일로 발송됩니다. 전달된 전자 메일의 링크를 통해 비밀번호를 변경하십시오. 단, 이는 FED 관리자 콘솔에서 관리하는 사용자 계정에 대해서만 적용됩니다. 조직이 Microsoft Active Directory (AD)와 같은 통합 디렉터리 서비스를 사용하는 경우, 해당 서비스에서 사용자 계정 비밀번호가 관리됩니다.
3. 관리자가 설정한 보안 정책에 따라 로그인 및 로그아웃 메뉴 사용이 불가능할 수 있습니다. DRM 클라이언트 설치 완료 후 PC에 로그인 시 DRM 클라이언트에 로그인하기 위한 사용자 ID와 비밀번호 입력 창이 자동으로 표시될 수 있습니다. 일부 사용자의 경우 로그인 창이 표시되지 않거나, 오프라인 상태 또는 네트워크에 연결되지 않은 상태에서만 로그인 창이 표시될 수 있습니다.

##### 보안 문서 아이콘
보안 문서는 파수의 FED 제품으로 암호화된 문서를 의미합니다. 암호화된 문서의 문서 아이콘에는 자물쇠 아이콘이 추가되며, 보안 문서 유형에 따라 서로 다른 색의 자물쇠 아이콘이 추가됩니다. 
| 보안 문서 유형 | 아이콘 | 설명 |
|----------------|--------|------|
| **FED-N** | ![](2025-11-10-10-25-13.png) **(초록색 자물쇠)** | FED-N 제품으로 암호화된 보안 문서에 추가되는 아이콘.<br>FED-N 문서는 사용자 PC 내에서 생성되며 FED-R 문서와는 달리 문서 등급의 적용을 받지 않으며, 연동 시스템에 설정되어 있는 접근 통제 수준에 따라 문서 사용 권한이 부여됩니다. |
| **FED-R** | ![](2025-11-10-10-25-28.png) **(주황색 자물쇠)** | FED-R 제품으로 암호화된 보안 문서에 추가되는 아이콘.<br>FED-R 문서는 사용자 PC 내에서 생성되며 FED-N 문서와는 달리 문서 등급의 적용을 받지 않으며, 연동 시스템에 설정되어 있는 접근 통제 수준에 따라 문서 사용 권한이 부여됩니다. |
| **FED-E** | ![](2025-11-10-10-25-40.png) **(보라색 자물쇠)** | FED-E 제품으로 암호화된 보안 문서에 추가되는 아이콘.<br>외부 공유를 위해 변환된 문서입니다.<br>FED-E 문서 사용 방법은 [외부 공유 문서 사용을 참고하십시오.](#외부-공유-문서-사용을-참고하십시오) |
| **MIP** |![](2025-11-10-10-26-50.png) **(파란색 자물쇠)** | Microsoft 365 제품을 편리하게 사용하면서도 FED가 제공하는 보안 수준을 동일하게 유지하기 위해 FED가 추가로 지원하는 유형의 보안 문서에 추가되는 아이콘.<br><br>FED-N, FED-R 또는 일반 문서가 MIP 문서로 변환될 수 있으며 ([MIP 문서 변환을 참고하십시오.](#mip-문서-변환을-참고하십시오)).<br>사용자는 FED 관련 정책에 따라 MIP 문서를 사용할 수 있습니다.<br>또한 MIP 문서 레이블 변경 등 세부적인 추가 기능은 Microsoft에서 제공하는 Microsoft Purview 공식 문서를 참고하시기 바랍니다.<br><br>참고: FED-N 문서와 다르게 문서 폐기, 임시 권한 설정, 외부 공유 문서 변환 기능은 지원되지 않습니다.
